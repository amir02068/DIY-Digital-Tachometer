# DIY Digital Tachometer

Arduino-based optical RPM meter using IR sensor and OLED display.

## Features
- Real-time RPM measurement
- Min / Max / Average RPM
- Optical IR sensor
- Portable (2×18650 battery)
- OLED 0.96" display

## Hardware
- Arduino Pro Mini
- LM358 Op-Amp
- IR LED + Photodiode
- OLED SSD1306
- 2×18650 Li-ion batteries

## Software
- Arduino IDE
- EasyEDA (PCB & schematic)
